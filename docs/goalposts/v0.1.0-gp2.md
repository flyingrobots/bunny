# Goalpost v0.1.0-GP2: Linear Algebra Primitives

## Description

Provide 2D and 3D vector representations using deterministic fixed-point coordinates in `bunny-linalg`.

## Status

Complete

## Acceptance Criteria

* `Vec2`/`Vec3` (floats) and `FixedVec2`/`FixedVec3` (fixed-point) structs defined.
* Vector addition, subtraction, negation, and scalar multiplication/division implemented.
* Dot products, cross products, and normalization functions implemented.
* Conversion traits (`From`/`Into`) implemented to map float vectors to fixed-point vectors.
* Integration tests verify all vector properties.
