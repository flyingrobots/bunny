# Goalpost v0.3.0-GP1: Stable BVH Tree

## Description

Build a memory-stable, zero-allocation bounding volume hierarchy (BVH) inside `bunny-broadphase`.

## Status

Completed

## Acceptance Criteria

* Stable array-backed tree layout requiring zero runtime heap allocation.
* SAH (Surface Area Heuristic) tree building solver.
* Deterministic tree traversal for box overlap and ray intersection.

## Completion Evidence

* `build_bvh` validates buffer sizes with checked arithmetic and uses checked
  node/primitive access helpers instead of unchecked indexing or `unwrap()`;
  malformed buffers return `None`.
* BVH traversal uses checked stack push/pop helpers and reports malformed node
  indices or primitive ranges through `TraversalError` instead of panicking.
* `crates/bunny-broadphase/tests/bvh_builder_tests.rs` covers malformed builder
  buffers in native and WASM tests.
* `crates/bunny-broadphase/tests/bvh_allocation_tests.rs` installs a native
  counting allocator and proves `build_bvh`, `intersect_aabb`, and
  `intersect_ray` perform zero heap allocations while operating on caller-owned
  buffers.
