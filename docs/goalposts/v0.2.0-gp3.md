# Goalpost v0.2.0-GP3: Closest Point Queries

## Description

Implement minimum-distance calculations between shapes in `bunny-query`.

## Status

Completed

## Acceptance Criteria

* Point-to-Triangle closest point calculation.
* Segment-to-Segment minimum distance calculation.
* AABB-to-Sphere closest point and overlap calculations.
* Validated with unit tests asserting byte-for-byte correctness.

## Completion Evidence

* `crates/bunny-query/tests/closest_raw_tests.rs` asserts raw Q32.32 integer
  outputs for AABB closest points, AABB/sphere overlap closest points,
  triangle face/edge/vertex closest points, and segment/segment closest pairs.
* The raw assertions include a half-unit projection to prove fractional
  fixed-point bytes, not only integer `to_f32()` projections.
* The test uses `wasm_bindgen_test(unsupported = test)`, so the byte-level
  assertions run in both native workspace tests and the WASM `bunny-query`
  gate.
