# Goalpost v0.2.0-GP1: Core Bounding Shapes

## Description

Implement core bounding envelopes and ray/sphere/box shapes using fixed-point vectors.

## Status

Completed

## Acceptance Criteria

* `FixedRay3`, `FixedAabb3`, and `FixedSphere3` structs are implemented in `bunny-geom` using `FixedVec3` coordinates.
* `From`/`Into` boundary mapping traits are defined to translate float geometries into fixed-point spaces.
* Integration tests verify bounding volumes under strict clippy denies.

## Completion Evidence

* Float ingress is explicit and validating: `Ray3::try_new`,
  `Aabb3::try_new`, `Sphere3::try_new`, and each type's `try_into_fixed`
  method reject non-finite coordinates, invalid AABB bounds, negative or
  non-finite sphere radii, and ray directions that cannot become fixed unit
  vectors.
* Fixed ingress remains available through `FixedRay3::try_from_float`,
  `FixedAabb3::try_from_float`, `FixedSphere3::try_from_float`, and the
  standard `TryFrom` implementations.
* Fixed-to-float egress is infallible and documented separately through
  `into_float` and the standard `From<Fixed*>` implementations.
* Integration tests cover valid round trips and every rejection path under
  strict clippy denies.
