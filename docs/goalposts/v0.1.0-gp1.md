# Goalpost v0.1.0-GP1: Deterministic Scalar Profile

## Description
Provide software-defined fixed-point math (`FixedQ32_32`) inside `bunny-num` to guarantee absolute bit-level CPU determinism across target platforms.

## Status
Complete

## Acceptance Criteria
* `FixedQ32_32` newtype wrapper exists.
* Standard arithmetic operators (`Add`, `Sub`, `Mul`, `Div`, `Neg` and assign variants) implemented.
* Banker's rounding (Ties-to-Even) implemented on division and multiplication.
* Saturating bounds applied for overflow protection.
* Deterministic integer `sqrt()` implemented and tested.
* Verified under Clippy and format checks.
