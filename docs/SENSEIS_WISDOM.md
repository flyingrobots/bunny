# Sensei's Wisdom™

Memorable rules for people who do not want to reread the standards doc before every PR.

1. **Bits beat vibes.** If two targets disagree, the algorithm is wrong.
2. **Floats are guests, not citizens.** They may enter at the border; they do not govern the core.
3. **A fixed-point wrapper with a public raw field is a costume.** Encapsulation, or go home.
4. **Saturation is not forgiveness.** It is a documented arithmetic policy or a bug generator.
5. **Degenerates are the API.** Zero area, collinear, coplanar, touching, empty, duplicate: test them.
6. **Ambient state is global mutable state wearing a fake mustache.** Inject clocks, seeds, and host access.
7. **Hash iteration order is not a personality.** Sort canonical outputs.
8. **A panic in library code is an undocumented trapdoor.** Return errors like an adult.
9. **Parallelism must preserve the story.** Same partitions, same reductions, same bits.
10. **`--no-verify` is a confession, not a strategy.** Fix the failure or write the exception down.
11. **Clippy nursery is a dojo, not a court.** Cherry-pick it. Do not crown it king.
12. **A commit is a receipt.** One cause, one effect, one explainable change.
